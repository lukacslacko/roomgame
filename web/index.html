Roomgame

Scan your room with the phone, then race a tiny vehicle around the voxelized model on the laptop.

Scanner (phone) Android Chrome with WebXR depth-sensing Recorder (phone) Capture frames + depth + pose for offline reconstruction Pose stability (phone) Tap origin · check return-to-zero drift Cube viewer (laptop) Lit, shadowed cubes · WASD fly cam Voxel viewer (laptop) 5 cm voxels · offline ray-march reconstruction Feature viewer (laptop) ORB feature triangulation · classical SfM-style Game / viewer (laptop) Renders the latest mesh (M3+)

The Python server prints a QR code on stderr when it starts — scan it with the phone to open the scanner directly.
